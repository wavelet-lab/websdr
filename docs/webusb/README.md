# WebSDR – SDR Interaction Subsystem

This documentation describes the **SDR interaction subsystem** of the WebSDR project.

## Key Features

- Frontend-only SDR architecture
- Direct WebUSB access to SDR hardware
- Device-agnostic control abstraction
- Pluggable SDR drivers (WASM or JavaScript)
- Real-time IQ streaming in the browser
